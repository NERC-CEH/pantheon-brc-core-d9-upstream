-- SUMMARY --

Transifex Connector is a module that builds atop of the Translation Management Suite (TMGMT)
allowing you to import content from your site to Transifex and retrieve it translated.

For a full description of the module, visit the project page:
  http://drupal.org/project/tmgmt_transifex

To submit bug reports and feature suggestions, or to track changes:
  http://drupal.org/project/issues/tmgmt_transifex


-- REQUIREMENTS --

* TMGMT

-- INSTALLATION --

* Install as usual, see https://www.drupal.org/docs/8/extending-drupal-8/installing-drupal-8-modules
  for further information.

-- CONFIGURATION --

* Configure the module at: Administration » Translation » Providers

  - Setup your Transifex API token.
  - Setup the target Transifex project.
  - Enable listening for webhooks by Transifex by setting a webhook secret
